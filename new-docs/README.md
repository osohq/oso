
## Development

To get started:

Install tailwind theme
```
npm install -g postcss postcss-cli autoprefixer
cd themes/tailwind
npm install
```

Install oso package
```
cd themes/oso
npm install
npm run-script build
```

Install hugo: https://gohugo.io/getting-started/installing

Ready to go!

```
hugo server
```
