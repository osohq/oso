into your local Ruby:

```
$ gem install oso-oso -v 0.9.0
```

or added to a projectâ€™s `Gemfile`:


**Requirements**

* Ruby version 2.4 or greater
* Supported platforms:
  * Linux
  * OS X
  * Windows
