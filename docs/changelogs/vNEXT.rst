New features
============

Modulo and remainder operators
------------------------------

The new ``mod`` and ``rem`` arithmetic operators can calculate moduli and
remainders, respectively:

.. code-block:: polar

  1 mod 3 == 1
  1 rem 3 == 1
  -1 mod 3 == 2
  -1 rem 3 == -1
