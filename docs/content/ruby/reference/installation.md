# Installation

The Ruby version of Oso is available on
[RubyGems](https://rubygems.org/gems/oso-oso) and can be installed to your
local Ruby:

```console
$ gem install oso-oso -v {{< version >}}
```

or added to a projectâ€™s `Gemfile`:

```ruby
gem 'oso-oso', '~> {{< version >}}'
```

## Requirements

- Ruby version 2.4 or greater
- Supported platforms:
  - Linux
  - macOS
  - Windows
