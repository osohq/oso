# Installation

The Rust version of Oso is available on
[crates.io](https://crates.io/crates/oso).

Add `oso` and `oso-derive` as dependencies in your `Cargo.toml`:

```toml
oso = "{{< version >}}"
oso-derive = "{{< version >}}"
```

## Requirements

- Supported platforms:
  - Linux
  - macOS
  - Windows
