## `oso` 0.26.4

### Python

#### Platform support

- We now publish wheels for Python 3.11 for all supported platforms.
  You should now be able to use `pip install oso` to get the
  latest Oso package.

  Thanks to [`@kkirsche`](https://github.com/kkirsche) help with this!
